# 高校学生社团服务管理一体化平台

## 项目背景

待开发的软件系统的名称：高校学生社团服务管理一体化平台

高校学生社团是学生自愿组成，为实现会员的共同愿望，按照其章程开展活动的非营利性群众组织。

中国中等学校和高等学校学生在自愿基础上自发组织而成、按照章程自主开展活动的学生群众组织。这些社团可打破年级、系科以及学校的界限。团结兴趣爱好相近的同学，发挥他们在某方面的特长，开展有益于学生身心健康的活动。

学生社团形式多种多样，如学术问题、社会问题的讨论研究会，文学艺术、体育、音乐、美术、影视等方面组成的活动小组，有文艺社、棋艺社、摄影社、美术社、歌唱队、话剧团、篮球队、足球队、数学社、物理社、化学社、微电影社团等等。

大学生社团是我国校园文化建设的重要载体，是我国高校第二课堂的引领者。每年各社团以其具有思想性、艺术性、知识性、趣味性、多样性的社团生活吸引广大学生积极参与其中。近年来各大高校在学生社团方面提供诸多支持，并取得了诸多成果。

然而在社团管理过程中，逐渐涌现出了许许多多难以解决的问题。

**第一，随着社团年限的增加，社团资料的保管变得越来越困难。**

年限增加势必会造成资料的堆积，而纸质资料在这种流动性较大的组织当中保管会更加困难。有的社团由于建社较早，许多以前的资料已经遗失；有的社团纸质资料较多，但是因为社团一年一换届，流动过大，因此保管已经非常困难；有的社团资料保管在学校相关部门，但是由于学校部门对于资料的清理等种种原因造成社团的资料损失严重。因此，如何保管各大高校的社团资料成为了摆在社团管理者面前的一个巨大难题。

**第二，手续过于繁琐，活动举办困难。**

随着对社团的管理加强，各大高校加强了对于学生社团活动的审核，因此对于普通社团来说，申办活动变得越来越繁琐。纸质资料东奔西跑签字盖章的情况在所难免，这边纸质材料还没有完成所有手续，那边社团活动已经马上就要开始了，甚至要面临着非法活动的困难。因此有效解决社团活动举办难度也是各大高校一直在探索的一个问题。

**第三，社团人员管理混乱，流入流出人员过多。**

对于社团来说，整体的社团管理人员流动会成为一个比较大的问题。各大社团目前对于这一问题尚无较好的解决方案。人员的管理混乱势必会带来更大的社团方面管理的问题和漏洞。

以上问题虽然并不会极大影响社团活动的开展，但是会极大影响社团活动质量和社团建设质量，对于社团的近期和长期发展都是很不利的。

因此，对于社团管理，现行的技术已经可以完成一个在线的数字化平台来对社团进行统一的管理和维护。经过我们的前期调研，社团的主要需求存在于：

1、能够在线申请社团，注销社团。

2、可以通过平台完成社团需要线下完成的各个手续。

3、可以利用平台对社团进行人员管理。

而这些，都是可以通过一个一体化平台来进行实现的。

随着学校的规模不断扩大，协会人数急剧增加，有关学生的各种信息也成倍增长。面对庞大的信息量，有必要开发社团信息管理系统来提高学团联对社团的管理工作的效率。

通过这样的系统，学团联和老师都可以做到对信息的规范管理、科学统计和快速查询，从而减少了管理方面的工作量。尤其是在今天数字管理盛行的时代，建立属于自己的实用性管理系统具有深远的意义。

基于此，信息与计算机工程学院 2016 级软件工程专业的学生破旧出新，大胆提出了建立高校学生服务管理一体化平台并进行设计，以期待利用统一综合的管理平台完成大学生学生社团的管理。项目组计划在设计完成之后完成接下来的总体设计，详细设计，编码，测试，并最终完成项目开发工作。

设计这个学生社团管理系统的初衷，是为了使学校能够更好地管理本学院繁多的学生社团，以及方便学生更好地了解各个学生社团的信息，从而丰富学生的课余生活。

高校学生社团服务管理一体化平台结合社团服务和社团管理为一体，一方面可以为学生社团提供消息发布平台，招新平台，另一方面可以作为校园数字化管理的一部分，方便学校相关主管部门对学生社团进行更加有效的管理。

## 运行环境

### 设备

该系统为 B/S 三层结构，它的运行环境分客户端、应用服务器端和数据库服务器端三部分。
以下是系统的软件环境。
（1）客户端
操作系统：Windows2000 Professional/XP 或更新版本。
浏览器：IE6 以上，其它常见浏览器如 FireFox。
（2）应用服务器端
操作系统：Windows2000 Server 或更新版本。
应用服务器：Tomcat 5.5 或更新版本。
数据库访问：JDBC。
（3）数据库服务器端
操作系统：Windows2000 Server 或更新版本。
数据库系统：SQLServer 2000 或更新版本。

**平台运行最低配置**
系统：Windows XP SP3/Windows Vista SP2/Windows 7
CPU：233MHZ 及以上
内存：1GB
硬盘：4GB 剩余空间
光驱：无特殊要求
显卡：ATI HD2600 XT 256MB 或者更高，Nvidia 7900GS 256MB 或更高，Core i3 或更高
声卡：无特殊需求
带宽：256kbps 及以上

**平台运行推荐配置**
系统：Windows Vista SP2/Windows 7/ Windows 8/ Windows 10
CPU：1.8 GHz Quad Core CPU
内存：4GB
硬盘：8GB 剩余空间
光驱：无特殊要求
显卡：ATI 4800se 512MB 或者更高，Nvidia9800se 512MB 或更高
声卡：电脑存在声卡
带宽：512kbps 及以上

### 支持软件

**平台开发环境：**
操作系统：Windows7 及以上
开发软件：IntelliJ IDEA+MySQL+ Navicat Premium+ WebStorm
**平台运行环境：**
操作系统：Windows7 及以上
平台平台基于 WEB 进行开发，需使用 IE6 以上和较新版本的网页浏览器正常访问。
**平台部署环境：**
服务器系统：Windows Server 2008 R2 及以上
部署软件：MySQL+Tomcat
